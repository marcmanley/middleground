# Frontend Testing Checklist

## Before editing

- [ ] Read `CLAUDE.md`
- [ ] Read `SKILL.md`
- [ ] Inspect the target page
- [ ] Inspect one comparable page
- [ ] Identify reusable selectors and patterns

## Desktop

- [ ] Navigation order is correct
- [ ] Headings and body alignment are correct
- [ ] Image crops are intentional
- [ ] Section spacing is deliberate
- [ ] Footer labels and paths are correct
- [ ] Console errors were reviewed

## Mobile

- [ ] Test at approximately 390px
- [ ] English paragraphs are left-aligned
- [ ] Arabic paragraphs remain RTL/right-aligned
- [ ] No horizontal overflow
- [ ] Important image content is not cropped
- [ ] Buttons and links remain easy to tap
- [ ] Footer remains readable

## Before finishing

- [ ] Analytics appears exactly once
- [ ] External links use safe target/rel attributes
- [ ] `Our Imām` is spelled and linked correctly
- [ ] New blog posts are added to the index
- [ ] Optimized images are used
- [ ] No unintended copy changes were made
- [ ] Every changed file is reported
